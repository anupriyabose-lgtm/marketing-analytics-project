# Marketing Analytics Project

## Overview
This project analyzes 12 months of digital marketing campaign data for a fictional D2C wellness brand.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## KPIs Analyzed
- CTR
- CVR
- CPC
- CAC
- ROAS

## Key Insights
- Email Marketing delivered the highest ROAS.
- SEO generated strong returns with low marketing spend.
- Influencer Marketing had the lowest ROI.
- Recommended reallocating budget to improve overall performance.
